Дан код: 
var a = 1, b = 1, c, d;
c = ++a; alert(c);               // 2		1+1=2 (начальное значение a + операция инкремента). После операции a=2, c=2.
d = b++; alert(d);               // 1		сначала происходит присваивание начального значения b, а затем операция инкремента. После операции b=2, d=2.
c = (2+ ++a); alert(c);          // 5		2+3=5 (2 + (операция инкремента + a со зачением 2). После операции a=3, c=5.
d = (2+ b++); alert(d);          // 4		2+2=4 (2 + b со значением 2). После присваивания d значения 4, производится операция инкремента и значение b равняется 3.
alert(a);                        // 3
alert(b);                        // 3

Почему код даёт именно такие результаты?
