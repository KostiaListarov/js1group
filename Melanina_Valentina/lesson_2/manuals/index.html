Задание 1

Дан код:

		var a = 1, b = 1, c, d;
		c = ++a; alert(c);               // 2
		d = b++; alert(d);               // 1
		c = (2+ ++a); alert(c);          // 5
		d = (2+ b++); alert(d);          // 4
		alert(a);                        // 3
		alert(b);                        // 3

Решение

№1 - Значение переменной "с" = 2 имеет префиксную форму, сначала увеличивается на 1, а потом возврашается.

№2 - Значение переменной "d" = 1, имеет постпрефиксную форму, сначала возвращает значение, а потом его увеличивает.

№3 - Значение переменной "с" = 5, имеет в формуле переменную "а" с префиксной формой, значение которой сначала увеличивается, а потом происходит сложение с 2. Формула имеет вид: с = (2 + (2 + 1)).

№4 - Значение переменной "d" = 4, имеет в формуле переменную "b" с постпрефиксной формулой, сначала выполняется действие по формуле и после этого значение "b" увеличивается. Формула имеет вид: с = (2 + 2).

№5 - В сообщении выходит значение переменной "a" = 3, т.к. она изначально была равна 1 и увеличивалась 2 раза.

№6 - В сообщении выходит значение переменной "d" = 3, т.к. она изначально была равна 1 и увеличивалась 2 раза.
